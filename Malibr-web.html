Faq

What is Malibr?

Malibr is a Social Media and Marketplace for creators and fans to sell, buy, collect and mint post NFTs

What is $MLB?

$MLB is the official cryptocurrency of Malibr for trading NFTs. The $MLB Token is a standard BEP-20 token of the Binance Smart Chain network with a limited supply.

Which blockchain network is Malibr based on?

Malibr is based on Binance Smart Chain, with the token type BEP-20
